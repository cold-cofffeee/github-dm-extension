# GitHub DM Extension - Icon Placeholder

This directory should contain the extension icons in the following sizes:
- icon16.png (16x16 pixels)
- icon48.png (48x48 pixels)
- icon128.png (128x128 pixels)

## Creating Icons

You can create icons using:
1. **Online tools**: 
   - https://www.favicon-generator.org/
   - https://realfavicongenerator.net/

2. **Design software**:
   - Adobe Photoshop
   - GIMP (free)
   - Figma (free)
   - Canva (free)

3. **AI tools**:
   - Use DALL-E, Midjourney, or similar to generate icons
   - Export in PNG format with transparent background

## Icon Design Tips

- Use GitHub's color scheme (green: #2ea44f)
- Include a message/envelope icon
- Keep it simple and recognizable
- Use transparent background
- Ensure it looks good at all sizes

## Temporary Solution

If you don't have icons yet, you can temporarily:
1. Remove icon references from manifest.json
2. Use placeholder images
3. Or download free icons from:
   - https://icons8.com/
   - https://www.flaticon.com/
